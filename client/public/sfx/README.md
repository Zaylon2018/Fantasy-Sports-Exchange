# Sound Effect Assets

This directory contains audio files for the cinematic onboarding experience.

## Required Files

### Crowd Cheer
**Filename:** `crowd_cheer.mp3`
- Duration: 5-10 seconds
- Content: Stadium crowd cheering, building excitement
- Volume: Starts low, builds to climax
- Format: MP3, 128kbps or higher

### Whoosh Sound
**Filename:** `whoosh.mp3`
- Duration: 0.5-1 second
- Content: Swoosh/whoosh sound for card movements
- Format: MP3, 128kbps or higher

### Pack Open Sound
**Filename:** `pack_open.mp3`
- Duration: 1-2 seconds
- Content: Magical/celebratory sound for pack opening
- Could be: sparkle, fanfare, magical burst
- Format: MP3, 128kbps or higher

## Placeholder/Temporary Assets

Until you have professional audio, you can:
1. Use silence (the app will work without audio)
2. Download free sounds from:
   - Freesound.org (free, CC licensed)
   - Zapsplat (free with attribution)
   - Mixkit (free, no attribution required)

## Browser Autoplay Policy

Audio must be triggered by user interaction:
- App shows "Tap to Start" button
- After tap, audio can play
- Initial state is muted
